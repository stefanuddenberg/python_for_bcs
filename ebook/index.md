# Preface

Welcome! This is the online textbook for the UIUC course BCOG 200/PSYC 496: Introduction to
Programming for Brain and Cognitive Science.

You can find the source code for this textbook, as well as the syllabus, on the
[course Github page](https://github.com/stefanuddenberg/python_for_bcs).

The book itself is organized into chapters, each of which has a number of sections. You can navigate
between chapters using the sidebar on the left, and sections using the table of contents on the
right.
